## Getting started

Clone the project and access the folder.

```bash
$ git clone https://github.com/Luis-Felipe-N/jokenpo.git
$ cd jokenpo
```

Follow the steps below:
```bash
# Install extension
$ Live Server

# Start the project
$ Go Live
```
The app will be available for access on your browser at http://127.0.0.1:5500/index.html
